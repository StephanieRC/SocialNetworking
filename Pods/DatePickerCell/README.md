## Installation

DatePickerCell is available through [CocoaPods](http://cocoapods.org). To install
it, add it to your Podfile:

```ruby
source 'https://github.com/CocoaPods/Specs.git'
platform :ios, '8.0'
use_frameworks!

pod 'DatePickerCell'
```
